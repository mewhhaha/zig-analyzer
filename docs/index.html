      Nothing here is mocked, but nothing runs live either: both servers were run over these files
      and their responses were recorded per position on 2026-07-17, against Zig 0.16.0 and
      ZLS 0.16.0
      with an explicit Zig executable, build-on-save enabled, and style warnings enabled. The
      capture sends a save event for every diagnostic fixture and waits up to 60 seconds for
      ZLS's build diagnostics.
    

    

      Re-capture with node docs/tools/capture_comparisons.js --repo . --zls /path/to/zls
      --zig /path/to/zig, or use Deno with the same arguments and
      --allow-read --allow-write --allow-run.
      The fixtures compile under
      zig build examples and live in
      examples/,
      with a manual reproduction guide.
