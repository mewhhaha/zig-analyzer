# Security policy

## Supported versions

Only the latest published zig-analyzer release receives security fixes. Each
release supports the exact Zig version printed by `zig-analyzer version`.

## Reporting a vulnerability

Report vulnerabilities privately through
[GitHub's security advisory form](https://github.com/mewhhaha/zig-analyzer/security/advisories/new).
Include the affected zig-analyzer version, operating system, reproduction
steps, and the impact you observed. Do not include sensitive details in a
public issue.

Reports remain private until a fix and release are available or coordinated
disclosure is otherwise agreed.
